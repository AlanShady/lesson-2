## Списки

Чтобы добавить ненумерованные списки, необзодимо пункты выделить звездочкой (*).
Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать. 
Напримр, вот так:
1. Первый пункт
2. Второй пункт
